# 간편 서버 (Simple Server)

로컬 폴더를 웹 서버로 열고, UPnP/포트 포워딩을 통해 인터넷에서도 접근할 수 있게 해주는 현대적인 파이썬 GUI 애플리케이션입니다.

## 주요 기능
- **원클릭 호스팅**: 로컬 네트워크에서 어떤 디렉토리든 즉시 서버로 열 수 있습니다.
- **UPnP 지원**: 공유기 설정을 수동으로 건드릴 필요 없이 자동으로 외부 접속용 포트 포워딩을 처리합니다 (UPnP 지원 공유기 필요).
- **현대적인 UI**: `customtkinter`를 사용한 세련되고 직관적인 디자인.
- **보안 중심**: 네트워크 보안 위험에 대한 명확한 경고와 면책 조항 포함.

## 면책 사항
**사용에 따른 책임은 사용자 본인에게 있습니다.**
포트 포워딩을 통해 기기를 외부 인터넷에 노출하는 것은 내재적인 보안 위험을 동반합니다. 개발자는 이 소프트웨어 사용으로 인해 발생하는 보안 침해, 데이터 손실 또는 기타 유무형의 피해에 대해 책임을 지지 않습니다.

## 요구 사항
- Python 3.x
- `requirements.txt`에 명시된 라이브러리들

## 설치 및 실행
1. 의존성 설치:
   ```bash
   pip install -r requirements.txt
   ```
2. 프로그램 실행:
   ```bash
   python main.py
   ```
   또는 제공된 스크립트 사용:
   ```bash
   ./run.sh
   ```

## 실행 파일 빌드
PyInstaller를 사용하여 단일 실행 파일로 패키징할 수 있습니다:
```bash
pyinstaller --noconsole --onefile main.py
```
